# 🍏 MacOS Web Desktop

A web-based simulation of the macOS desktop, built with React, Tailwind, and Framer Motion. Includes features like wallpaper changing, app-like windows (e.g. Settings), and dock interactions.

---

## 🛠️ Tech Stack

- ⚛️ [React](https://reactjs.org)
- 🎨 [Tailwind CSS](https://tailwindcss.com)
- 💨 [Framer Motion](https://www.framer.com/motion/)
- 🌿 [Zustand](https://github.com/pmndrs/zustand)
- ⚡ [Vite](https://vitejs.dev) for blazing-fast dev environment

---

## 🚀 Getting Started

```bash
# 1. Clone the repo
git clone https://github.com/yourusername/macos-web-desktop.git
cd macos-web-desktop

# 2. Install dependencies
npm install

# 3. Run the dev server
npm run dev
